# LTI Show 페이지와 Canvas Show 페이지 UI 싱크 작업 계획

**작성일**: 2026-01-10
**목적**: LTI 프로젝트 상세 페이지를 Canvas 원본 UI와 정확히 일치시키기
**작업 방식**: 단계별 진행, 각 단계마다 MD 파일에 흔적 남기기

---

## 📋 현재 상황 분석

### Canvas Show 페이지 구조 (hy_project_show 분석)

**레이아웃 구조**:
1. **AssignmentGroupHeader** (공통)
   - 프로젝트 제목
   - 프로젝트 정보 카드:
     - Group Assignment
     - Group Set
     - Slack Channel
   - Step Navigator (가로 스크롤 가능한 Step 카드들)

2. **AssignmentGroupContent** (교수용)
   - Step 상세 제목: "Step 1: title"
   - 그룹 선택 드롭다운 (여러 그룹이 있을 경우)
   - Step 상세 카드:
     - Description (Rich Text Editor 출력)
     - Points, Grades Assignment, Submitting, Allowed Attempts
   - To Do 섹션 (Task 관리, 가로 스크롤)

3. **StudentAssignmentGroupContent** (학생용)
   - Step 상세 제목: "Step 1 title"
   - Step 상세 카드:
     - Description (Rich Text Editor 출력)
     - Points, Due, Available, Submitting, File Types
   - To Do 섹션 (Task 관리, 가로 스크롤)

4. **AssignmentSubmissions** (교수용)
   - Evaluations 섹션
   - 그룹 멤버별 제출 상태 (Graded/Submitted/Not Yet)
   - SpeedGrader 링크
   - Assign Peer Review 버튼

5. **StudentAssignmentSubmissions** (학생용)
   - Submission Phase 섹션
   - 3단계 카드 레이아웃:
     - 1. Submit (제출)
     - 2. Review (Peer Review, 조건부)
     - 3. Feedback (피드백 확인)

### LTI Show 페이지 현재 구조

**레이아웃**:
- 프로젝트 헤더 (제목, 수정/삭제 버튼)
- Step 카드들 (세로 나열)
  - Step 제목, 상태 뱃지
  - Description
  - Step Details (Available from, Due Date, Until, Points, Status, Submission Types)
  - 학생/교수 분기:
    - 교수: Submission Statistics (통계 카드)
    - 학생: Your Submission (제출 상태)

**차이점**:
- ❌ Step Navigator 없음 (가로 스크롤 Step 카드)
- ❌ 프로젝트 정보 카드 없음 (Group Assignment, Group Set, Slack Channel)
- ❌ To Do 섹션 없음 (Task 관리)
- ❌ Submission Phase 섹션 없음 (3단계 카드 레이아웃)
- ❌ Evaluations 섹션 없음 (교수용 그룹 멤버 제출 상태)
- ⚠️ 학생/교수 분리는 있지만 UI 구조가 다름

## 🎯 목표 및 제약사항

### 목표
1. Canvas Show 페이지와 **시각적으로 동일**하게 만들기
2. 학생/교수용 뷰를 Canvas와 동일한 구조로 분리
3. 모든 주요 섹션 구현 (Step Navigator, Submission Phase, To Do 등)

### 제약사항
1. **To Do 섹션**: Task 관리 기능이 LTI에 구현되어 있는지 확인 필요
2. **Submission Phase**: Canvas API로 제출 기능 구현 필요
3. **Peer Review**: Canvas API로 Peer Review 기능 구현 필요
4. **React vs ERB**: Canvas는 React, LTI는 ERB 템플릿 (구조만 일치시킴)

| **Step 상세** | | | |
| Step 제목 | ✅ | ✅ | ✅ 일치 |
| Description | ✅ (Rich Text) | ✅ (plain text) | ⚠️ 형식 차이 |
